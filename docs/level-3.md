## How TaskFlow Pro Covers Level 3

TaskFlow Pro includes a full backend system using Node.js, Express, MongoDB, and JWT.

Features include:

- User registration
- User login
- Password hashing with bcrypt
- JWT protected routes
- Project CRUD API
- Task CRUD API
- MongoDB Atlas database
- Change password feature
