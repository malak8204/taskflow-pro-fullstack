## How TaskFlow Pro Covers Level 2

TaskFlow Pro includes a React-based task management interface.

Features include:

- Create tasks
- Edit tasks
- Delete tasks
- Filter tasks by status
- Search tasks
- Add due dates
- Show priority badges
- Manage React state with hooks
- Connect frontend to backend using Axios
