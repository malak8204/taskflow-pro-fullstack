# TaskFlow Pro Development Steps

## Step 1: Project Setup

Created the main project structure with client, server, and docs folders.

## Step 2: Backend Setup

Initialized the backend using Node.js and installed Express, Mongoose, dotenv, cors, bcryptjs, jsonwebtoken, and nodemon.

## Step 3: MongoDB Atlas

Created a MongoDB Atlas database and connected it to the backend using Mongoose.

## Step 4: Authentication

Implemented register, login, password hashing, JWT token generation, and protected middleware.

## Step 5: Project CRUD

Created backend routes for creating, reading, updating, and deleting projects.

## Step 6: Task CRUD

Created backend routes for creating, reading, updating, and deleting tasks.

## Step 7: React Frontend

Created the frontend using React and Vite.

## Step 8: Frontend Pages

Created Home, Login, Register, Dashboard, Projects, Tasks, and Profile pages.

## Step 9: Dashboard

Added dashboard statistics showing total projects, tasks, completed tasks, pending tasks, and in-progress tasks.

## Step 10: Extra Features

Added project search, task search, task filtering, due dates, priority badges, and change password functionality.

## Step 11: GitHub

Initialized Git, committed the code, connected the project to GitHub, and pushed the repository.
