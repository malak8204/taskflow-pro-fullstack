# TaskFlow Pro

TaskFlow Pro is a full-stack project and task management web application built with React, Node.js, Express, MongoDB, and JWT authentication.

## Features

- User registration and login
- JWT authentication
- Protected routes
- Dashboard statistics
- Project CRUD operations
- Task CRUD operations
- Task search and filtering
- Project search
- Task due dates
- Priority badges
- Change password feature
- Responsive user interface

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- dotenv
- cors

## How to Run Locally

### Backend

cd server  
npm install  
npm run dev  

Backend runs on:

http://localhost:5000

### Frontend

cd client  
npm install  
npm run dev  

Frontend runs on:

http://localhost:5173
